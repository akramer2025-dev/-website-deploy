# 🚀 موقع أكرم مصطفى - خبير التسويق الإلكتروني

موقع شخصي احترافي لعرض خدمات ومحاضرات التسويق الإلكتروني والسوشيال ميديا.

- 📱 **متجاوب تماماً** يعمل على جميع الأجهزة (موبايل، تابلت، كمبيوتر)
- 🌟 **تأثيرات حركية متقدمة** باستخدام AOS و Particles.js
- 🎯 **تحسين محركات البحث SEO** جاهز
- ⚡ **أداء عالي وسرعة تحميل ممتازة**
- 🔗 **روابط السوشيال ميديا** لجميع المنصات الرئيسية

## 🚀 كيفية الاستخدام

### 1. فتح الموقع:
- افتح ملف `index.html` في المتصفح مباشرة
- أو استخدم Live Server في VS Code

### 2. تخصيص المحتوى:

#### تغيير المعلومات الشخصية:
- افتح `index.html`
- ابحث عن الأقسام التالية لتعديلها:
  - **قسم البطل (Hero)**: غير العنوان والوصف
  - **قسم من أنا (About)**: أضف معلوماتك الشخصية
  - **التواصل (Contact)**: غير البريد الإلكتروني ورقم الهاتف

#### إضافة صورتك الشخصية:
1. احفظ صورتك في مجلد `images/`
2. في `index.html`، ابحث عن:
```html
<div class="image-placeholder">
    <i class="fas fa-user"></i>
</div>
```
3. استبدلها بـ:
```html
<img src="images/your-photo.jpg" alt="أكرم مصطفى">
```

#### تحديث روابط السوشيال ميديا:
ابحث عن `<a href="#"` واستبدل `#` بروابطك الحقيقية:
```html
<a href="https://facebook.com/yourpage" class="social-link facebook">
```

### 3. إضافة كورسات جديدة:

في قسم الكورسات، انسخ أي كارت كورس والصق نسخة جديدة:
```html
<div class="course-card" data-aos="fade-up">
    <div class="course-icon facebook">
        <i class="fab fa-facebook-f"></i>
    </div>
    <h3>اسم الكورس الجديد</h3>
    <p>وصف الكورس...</p>
    <!-- ... باقي المحتوى -->
</div>
```

## 🎯 التخصيص المتقدم

### تغيير الألوان:
افتح `css/style.css` وعدل المتغيرات في البداية:
```css
:root {
    --primary-blue: #0a1f44;
    --gradient-pink: #e94e77;
    --gradient-orange: #f7931e;
    /* ... */
}
```

### تعديل التأثيرات الحركية:
في `js/script.js`، عدل إعدادات AOS:
```javascript
AOS.init({
    duration: 1000,  // مدة الحركة بالميلي ثانية
    easing: 'ease-in-out',  // نوع الحركة
    once: true,  // تشغيل مرة واحدة فقط
    offset: 100  // المسافة قبل بدء الحركة
});
```

### تعديل الجزيئات (Particles):
في `js/script.js`، عدل إعدادات particlesJS:
```javascript
particles: {
    number: {
        value: 80,  // عدد الجزيئات
        // ...
    }
}
```

## 📧 نموذج التواصل

حالياً، النموذج يعرض رسالة نجاح فقط. لتفعيله:

### الخيار 1: استخدام EmailJS (مجاني):
1. سجل على [emailjs.com](https://www.emailjs.com/)
2. أنشئ قالب بريد إلكتروني
3. أضف الكود في `js/script.js`

### الخيار 2: استخدام Formspree:
1. سجل على [formspree.io](https://formspree.io/)
2. أضف endpoint في form action

### الخيار 3: Backend مخصص:
استخدم Node.js, PHP, أو أي Backend آخر
