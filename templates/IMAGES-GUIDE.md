# 📸 دليل إضافة صور القوالب

## لماذا نحتاج الصور؟

حالياً الموقع يستخدم صور Placeholder مؤقتة. لتحسين الموقع وجعله أكثر احترافية، يجب إضافة صور حقيقية للقوالب.

---

## 🎯 الصور المطلوبة لكل قالب

### 1. صورة رئيسية كبيرة (Hero Image)
- **الأبعاد:** 800x600 بكسل
- **الاستخدام:** تظهر في بطاقة القالب والنافذة المنبثقة
- **المحتوى:** لقطة شاشة للصفحة الرئيسية للقالب

### 2. صورة مصغرة (Thumbnail)
- **الأبعاد:** 400x300 بكسل
- **الاستخدام:** معاينة سريعة في الشبكة
- **المحتوى:** نفس الصورة الرئيسية بحجم أصغر

### 3. معرض صور (Gallery) - اختياري
- **الأبعاد:** 1200x900 بكسل
- **العدد:** 4-8 صور
- **المحتوى:** 
  - الصفحة الرئيسية
  - صفحة من نحن
  - صفحة الخدمات
  - صفحة المدونة
  - صفحة التواصل
  - صفحات أخرى مهمة

## 🛠️ كيفية الحصول على الصور

### الطريقة 1: لقطات شاشة من الموقع التجريبي

1. **افتح الموقع التجريبي**
   ```
   https://xtratheme.com
   ```

2. **خذ لقطات شاشة احترافية**
   - استخدم أداة مثل [Screely](https://screely.com) لإضافة إطار
   - أو استخدم [Browser Frame](https://browserframe.com)

3. **احفظ الصور**
   - احفظ بصيغة JPG للحجم الأصغر
   - أو PNG للجودة الأعلى

### الطريقة 2: من موقع البائع

1. **زر موقع الثيم**
   ```
   https://xtratheme.com
   ```

2. **ابحث عن Screenshots أو Gallery**
   - عادة تكون في صفحة المنتج
   - أو في صفحة Demo

3. **حمّل الصور**
   - انقر بزر الماوس الأيمن → Save image as
   - احفظها في مجلد images الخاص بالثيم

### الطريقة 3: تنصيب محلي

1. **نصّب القالب محلياً**
   - استخدم Local by Flywheel أو XAMPP
   - نصّب WordPress + القالب

2. **استورد قالب جاهز**
   - من لوحة تحكم القالب
   - استورد أحد التصاميمالجاهزة

3. **خذ لقطات شاشة**
   - بدقة عالية
   - لصفحات مختلفة

## ⚡ تحسين الصور قبل الرفع

### استخدم أدوات ضغط الصور:

1. **[TinyPNG](https://tinypng.com)**
   - ضغط ممتاز مع الحفاظ على الجودة
   - يدعم JPG و PNG

2. **[Squoosh](https://squoosh.app)**
   - أداة Google للضغط
   - تحكم كامل في الإعدادات

3. **[ImageOptim](https://imageoptim.com)** (Mac)
   - تطبيق سطح مكتب
   - ضغط متقدم

### معايير الضغط:
- **الجودة:** 80-85% (توازن بين الجودة والحجم)
- **الحجم المستهدف:** 
  - Hero Image: أقل من 200 KB
  - Thumbnail: أقل من 50 KB
  - Screenshots: أقل من 300 KB لكل صورة

## 🔄 تحديث الموقع بالصور الجديدة

### 1. رفع الصور

ارفع الصور إلى مجلد مخصص:
```
/images/templates/xtra/
├── hero.jpg
├── thumbnail.jpg
├── screenshot-1.jpg
├── screenshot-2.jpg
└── ...
```

### 2. تحديث ملف JavaScript

عدّل ملف `js/ready-websites.js`:

```javascript
{
    id: 'xtra-theme',
    name: 'Xtra Premium',
    // ...
    image: '/images/templates/xtra/hero.jpg',              // ⬅️ غيّر هنا
    thumbnail: '/images/templates/xtra/thumbnail.jpg',     // ⬅️ وهنا
    // ...
}
```

### 3. إضافة معرض الصور (اختياري)

أضف خاصية `gallery` للقالب:

```javascript
{
    id: 'xtra-theme',
    name: 'Xtra Premium',
    // ... باقي الخصائص
    gallery: [
        '/images/templates/xtra/screenshot-1.jpg',
        '/images/templates/xtra/screenshot-2.jpg',
        '/images/templates/xtra/screenshot-3.jpg',
        '/images/templates/xtra/screenshot-4.jpg'
    ]
}
```

## 🎨 نصائح لصور احترافية

### ✅ افعل:
1. **استخدم دقة عالية** - الصور الواضحة تعطي انطباع احترافي
2. **أظهر التنوع** - صور لصفحات مختلفة ووظائف متعددة
3. **حافظ على الاتساق** - نفس أبعاد الصور لكل القوالب
4. **اعرض المميزات** - ركز على المميزات الفريدة للقالب
5. **أضف سياق** - أظهر القالب في سياق حقيقي

### ❌ تجنب:
1. **الصور المشوشة** - تعطي انطباع غير احترافي
2. **الصور الصغيرة جداً** - صعبة المشاهدة
3. **الصور الكبيرة جداً** - تبطئ الموقع
4. **نفس الصورة مكررة** - مملة للزائر
5. **محتوى غير ملائم** - تأكد أن المحتوى مناسب
