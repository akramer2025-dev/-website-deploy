v0.dev - موقع في ثواني ⚡

🚀 الرابط المباشر:
v0.dev

📝 كيف تستخدمه؟

  • افتح v0.dev
  • سجل دخول بحساب GitHub (مجاني)
  • اكتب Prompt واضح بالإنجليزية:
    "Create a modern landing page for a pizza restaurant.
    Include: hero section, menu, about us, contact form.
    Use red and yellow colors. Add a CTA button 'Order Now'."
  • اضغط Generate
  • انتظر 10-30 ثانية ⏱️
  • شاهد 3 نسخ مختلفة من الموقع
  • اختار الأفضل → Edit لو عايز تعدل
  • حمّل الكود → Export Code
✅ المميزات:
  • كود نظيف واحترافي (React + Tailwind CSS)
  • Responsive (يشتغل على الموبايل)
  • سريع جداً
💰 السعر: مجاني ل 200 credit شهرياً (10-20 موقع)

SEO - اظهر في جوجل 🔍

🎯 أساسيات SEO للموقع:

  • Title Tag: عنوان الصفحة (50-60 حرف)
    <title>بيتزا توب - أفضل بيتزا في القاهرة</title>
  • Meta Description: وصف الصفحة (150-160 حرف)
    <meta name="description" content="توصيل بيتزا سريع في القاهرة خلال 20 دقيقة. اطلب الآن!">
  • Keywords: الكلمات المفتاحية في المحتوى
  • Alt Text للصور: وصف الصورة
    <img src="pizza.jpg" alt="بيتزا مارجريتا طازجة">
  • سرعة الموقع: استخدم أدوات ضغط الصور
  • Mobile-Friendly: تأكد إن الموقع يشتغل على الموبايل
🛠️ أداة فحص SEO مجانية:
Google PageSpeed Insights
