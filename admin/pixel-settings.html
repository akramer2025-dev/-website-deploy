تحديث Pixel ID

كيف تحصل على Pixel ID؟

  1. اذهب إلى Facebook Events Manager
  2. اختر موقعك أو أنشئ Pixel جديد
  3. انسخ رقم الـ Pixel ID (مكون من 15-16 رقم)
  4. الصقه في الخانة أعلاه واضغط حفظ
  5. بعد الحفظ، ارفع التحديثات على Git لتفعيل البيكسل
فتح Events Manager

خطوة أخيرة: تحديث الكود

افتح الملف التالي وحدث الـ Pixel ID:

js/meta-pixel.js

ابحث عن السطر:

var PIXEL_ID = 'القيمة_القديمة';

واستبدله بـ:

var PIXEL_ID = 'YOUR_NEW_ID';

ثم:

  1. احفظ الملف
  2. ارفع التحديثات على Git: git add js/meta-pixel.js && git commit -m "Update pixel ID" && git push
  3. تحقق من Meta Events Manager بعد بضع دقائق
