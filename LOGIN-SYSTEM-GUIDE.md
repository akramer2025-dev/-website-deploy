# 🔐 دليل نظام تسجيل الدخول والمحاضرات المسجلة

## نظرة عامة

تم إضافة نظام تسجيل دخول كامل لحماية المحاضرات وإدارة صلاحيات الطلاب. النظام يسمح لكل طالب برؤية المحاضرات المسجلة المتاحة له فقط.

## 🎯 المميزات

### 1. نظام تسجيل الدخول
- شاشة دخول آمنة لكل طالب
- حفظ الجلسة لمدة 24 ساعة
- رسائل خطأ واضحة عند إدخال بيانات خاطئة

### 2. صفحة المحاضرات المسجلة
- عرض المحاضرات المتاحة للطالب فقط
- بطاقات تفاعلية لكل محاضرة
- معلومات تفصيلية (عدد الأقسام، زمن القراءة)

### 3. حماية المحتوى
- التحقق من صلاحيات الوصول لكل محاضرة
- منع الوصول غير المصرح للمحاضرات
- نظام جلسات آمن

### 4. تجربة مستخدم متكاملة
- ترحيب بالاسم في كل صفحة
- زر تسجيل خروج في كل مكان
- حفظ تلقائي لتقدم الطالب

## 📝 طريقة الاستخدام

### للطالب:

1. **الدخول للموقع:**
   - افتح رابط: `https://www.akrammostafa.com/lecture-viewer-v2.html`
   - ستظهر شاشة تسجيل الدخول

2. **تسجيل الدخول:**
   - أدخل اسم المستخدم وكلمة المرور
   - اضغط زر "دخول"

3. **مشاهدة المحاضرات المسجلة:**
   - بعد الدخول، ستظهر جميع المحاضرات المتاحة لك
   - اضغط على أي محاضرة لبدء المشاهدة

4. **العودة لقائمة المحاضرات:**
   - اضغط على اسمك في الأعلى
   - أو احذف `?id=...` من الرابط

5. **تسجيل الخروج:**
   - اضغط على زر "تسجيل الخروج" في أي صفحة

### للمدير (إضافة طلاب جدد):

افتح ملف `lecture-viewer-v2.html` وابحث عن:

```javascript
const STUDENTS_DB = {
```

وأضف حساب جديد بهذا الشكل:

```javascript
'username': {
    password: 'password123',
    name: 'اسم الطالب',
    email: 'email@example.com',
    allowedLectures: ['lec_fb_1', 'lec_april_1']
}
```

## 🔒 الأمان

⚠️ **مهم:** النظام الحالي يعمل على localStorage وهو مناسب للتطوير والاختبار فقط.

للاستخدام الفعلي في الإنتاج، يُنصح بـ:

- تشفير كلمات المرور
- استخدام JWT tokens للجلسات
- إضافة HTTPS للموقع

## 🎨 التخصيص

### إضافة محاضرة جديدة:

1. أضف بيانات المحاضرة في `HARDCODED_LECTURES`
2. أضف معلوماتها في `LECTURES_CATALOG`
3. أضف ID المحاضرة في `allowedLectures` للطلاب المطلوبين

### تعديل مدة الجلسة:

ابحث عن هذا السطر:
```javascript
expires: new Date().getTime() + (24 * 60 * 60 * 1000) // 24 ساعة
```

وغيّر الرقم حسب الحاجة.
